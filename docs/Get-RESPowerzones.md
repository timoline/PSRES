# Get-RESPowerzones

## SYNOPSIS
Shows the powerzones from RES Workspace Manager / Powerfuse

## SYNTAX

```
Get-RESPowerzones [[-Name] <String>] [[-Rule] <String>] [[-Guid] <String>] [-Disabled] [<CommonParameters>]
```

## DESCRIPTION
Shows the powerzones from RES Workspace Manager / Powerfuse

## EXAMPLES

### EXAMPLE 1
```
Get-RESPowerzones
```

Shows the powerzones
